How to Use:

Enter your last.fm username, or upload a JSON file
that you have downloaded previously from this site.
Loading will commence.

If the loading progress pauses for too long,
download your JSON file, refresh the page, and
upload it to resume from where loading left off.

Once loaded, you can click on each album/artist
to see its songs/albums. You are also able to drag
and scroll. Press Ctrl+Shift+Z to zoom back out,
or use the "Zoom Out" button.

Make sure to download the JSON file once loading
is finished to skip the wait next time!
